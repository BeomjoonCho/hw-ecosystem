# 기술 파트너 대시보드 화면 구성 계획

## 개요
스크린샷을 기반으로 한 "기술 파트너 대시보드" 화면 구성 및 개발 계획

## 화면 구조

### 1. 헤더 영역 (Header)
- **제목**: "기술 파트너 대시보드"
- **검색 영역**:
  - 통합 검색 입력 필드 (placeholder: "기술,파트너 통합 검색")
- **네비게이션 링크**:
  - "전체"
  - "파트너"
  - "기술 특성"
  - "Key Player"

### 2. 메인 콘텐츠 영역

#### 2.1 "기술 파트너 목록" 섹션
- **제목**: "기술 파트너 목록"
- **액션 버튼**: "파트너 기술 제안" (보라색 버튼)
- **파트너 카드 그리드** (2x2):
  - 각 카드 정보:
    - 회사명
    - 설명
    - 주요 기술 태그
    - 핵심 기술
    - 한화 계열 수행 (건수)
    - 전체 수행 (건수)
    - 기술 점수 (Technology Score) /10
    - 시장 점수 (Market Score) /10
    - "상세보기" 버튼

**파트너 카드 예시**:
1. **Cohere**
   - 설명: Enterprise LLM - RAG Platform
   - 주요 기술: LLM / RAG / Embeddings
   - 기술 점수: 4.3/10
   - 시장 점수: 5/10

2. **Databricks**
   - 설명: Manufacturing Data Platform
   - 주요 기술: Lakehouse / AI Data Points
   - 기술 점수: 4.7/10
   - 시장 점수: 5/10

3. **Hexagon**
   - 설명: Digital Twin - Smart Factory
   - 주요 기술: 3D Twin / Manufacturing Analysis
   - 기술 점수: 4.4/10
   - 시장 점수: 5/10

4. **KOPENS**
   - 설명: Industrial IIoT - Digital Twin Platform
   - 주요 기술: IIoT Platform / Edge Computing / Digital Twin
   - 기술 점수: 4.7/10
   - 시장 점수: 5/10

#### 2.2 "Technology Intelligence" 섹션
- **제목**: "Technology Intelligence"
- **액션**: "더보기" 링크
- **3개 컬럼 레이아웃**:

  1. **"최신 기술 파트너" 컬럼**
     - 파트너 목록 (회사명 + 설명)
     - 예: Cohere, Enterprise LLM - RAG Platform
     - 예: Databricks, Manufacturing Data Platform

  2. **"시장 동향" 컬럼**
     - 실제 기사 링크 리스트 (2026년 기사 4개)
     - 각 항목은 클릭 가능한 링크로 표시
     - 예: AI Investment Boom to Lift U.S. Corporate Bond Issuance to $2.46 Trillion in 2026 (2026.01)
     - 예: Onix 2026 AI Trends: Multi-agent Systems Redefine Enterprise Workflows (2026.01)
     - 예: VCs Predict Tiny Teams, Personal Agents, ROI-Focus as Key Tech Trends in 2026 (2026.01)
     - 예: South Korea's Push for a Native AI Model Faces Controversy (2026.01)

  3. **"컨퍼런스 동향" 컬럼**
     - 실제 컨퍼런스 링크 리스트 (2026년 개최/예정)
     - 각 항목은 클릭 가능한 링크로 표시
     - 예: NVIDIA GTC 2026 (2026.03)
     - 예: AI Impact Summit 2026 (2026.02)
     - 예: Gartner Data & Analytics Summit 2026 (2026.03)
     - 예: All Things AI 2026 (2026.03)

## 파트너 상세보기 페이지

### 페이지 구조
- URL: `/partners/[id]`
- 모달이 아닌 별도 페이지로 구현
- thevc.kr/upstage 스타일 참고

### 탭 구성
1. **기본 정보** (thevc.kr 스타일로 확장)
   - 기업 개요
     - 회사명, 설립년도, 본사 위치, 대표자
     - 유사 기업 목록
   - 연관 키워드
   - 주요 제품/서비스
   - 핵심 기술 및 주요 기술 태그
   - 한화 계열 수행 건수 / 전체 수행 건수
   - 최근 뉴스
   - 투자 유치 정보
   - 국가 R&D 프로젝트
   - 종합 평가 (기술 점수, 시장 점수)
   - 주요 정보 카드 (상태, 업력, 임직원 수, 특허)

2. **수행 목록**: 한화 계열사별 수행 프로젝트 목록 (프로젝트명, 회사, 기간, 상태)

3. **기술 점수**: 기술 우수성 평가 기준
   - A. 기술 완성도
     - A1. 기능 충족도
     - A2. 커스터마이징 용이성
     - A3. 신뢰성/보안성
   - B. 비즈니스 적합성
     - B1. 서비스 다양성/혁신성
     - B2. 글로벌 지원
     - B3. 적용 사례
   - 종합 기술 점수

4. **시장 점수**: 파트너십 역량 평가 기준
   - C. 파트너십 효율성
     - C1. 벤더 재무건전성 및 명성
     - C2. 교육 및 지원
     - C3. 비용 효율성
   - D. 전략적 가치
     - D1. 신기술 연계
     - D2. 소스코드 공유
     - D3. 협업 의지
   - 종합 시장 점수

## 타입 정의

```typescript
// Partner 타입 (thevc.kr 참고하여 확장)
interface Partner {
  id: string;
  name: string;
  description: string;
  technologies: string[];
  coreTechnology: string; // 핵심 기술
  projectCount: number; // 한화 계열 수행 건수
  totalProjectCount: number; // 전체 수행 건수
  technologyScore: number;
  marketScore: number;
  // 기본 정보 확장 필드
  website?: string; // 홈페이지
  foundedYear?: number; // 설립년도
  location?: string; // 본사 위치
  ceo?: string; // 대표자
  employeeCount?: number; // 임직원 수
  status?: string; // 상태 (상장/비상장)
  businessYears?: number; // 업력
  patents?: number; // 특허 수
  similarCompanies?: string[]; // 유사 기업
  keywords?: string[]; // 연관 키워드
  products?: Array<{
    name: string;
    description?: string;
  }>; // 주요 제품/서비스
  recentNews?: Array<{
    title: string;
    date: string;
    source?: string;
  }>; // 최근 뉴스
  investmentRounds?: Array<{
    round: string;
    date: string;
    amount?: string;
  }>; // 투자 유치
  rndProjects?: Array<{
    title: string;
    category: string;
    period: string;
  }>; // 국가 R&D
}

// Technology Intelligence 타입
interface TechnologyIntelligence {
  latestPartners: Array<{
    name: string;
    description: string;
  }>;
  marketTrends: Array<{
    title: string;
    url: string;
    date?: string;
    source?: string;
  }>; // 시장 동향 - 실제 기사 링크
  conferenceTrends: Array<{
    title: string;
    url: string;
    date?: string;
  }>; // 컨퍼런스 동향 - 실제 컨퍼런스 링크
}
```

## 파트너 기술 제안 기능

### 기능 개요
- "파트너 기술 제안" 버튼 클릭 시 팝업 모달 표시
- 업체 기본 정보 및 상세 내용 입력 폼 제공
- 프로세스 진행 현황 표시

### 프로세스 진행 현황
1. **업체 입력**: 사용자가 업체 정보를 입력하는 단계
2. **내부 검토 및 평가** (이노베이션 Lab): 제안된 업체를 내부에서 검토하는 단계
3. **구매팀 연계**: 검토 완료 후 구매팀으로 전달되는 단계

### 입력 폼 구성
- **업체 기본 정보**:
  - 업체명 (필수)
  - 업체 홈페이지
  - 대표자명
  - 설립년도
  - 본사 위치
  - 주요 기술/제품
  - 핵심 기술 설명
- **상세 내용**:
  - 제안 배경 및 목적
  - 기술적 우위성
  - 적용 가능 분야
  - 기대 효과
  - 참고 자료 (링크 등)
  - 기타 의견

### 구현 방식
- Dialog 컴포넌트 사용
- React Hook Form으로 폼 관리
- 프로세스 진행 현황은 단계별로 시각화 (현재 단계 강조)

## 키워드 필터링 기능

### 기능 개요
- 파트너 카드의 주요 기술 태그(LLM, RAG 등) 클릭 시 해당 키워드를 가진 파트너 목록 표시
- 상세 페이지의 연관 키워드('OCR', 'AI통합솔루션' 등) 클릭 시 해당 키워드를 가진 파트너 목록 표시

### 구현 방식
- **URL 구조**: `/partners?keyword=LLM` (쿼리 파라미터 사용)
- **검색 범위**: 
  - `technologies` 배열 (주요 기술 태그)
  - `keywords` 배열 (연관 키워드)
- **필터링 로직**: 대소문자 구분 없이 부분 일치 검색
- **표시 방식**: 필터링된 파트너 목록을 PartnerGrid 컴포넌트로 표시

### 페이지 구조
- **URL**: `/partners` (쿼리 파라미터: `?keyword=xxx`)
- **레이아웃**: 메인 페이지와 동일한 구조
- **헤더**: "키워드: [키워드명]" 형태로 표시
- **필터링 결과**: 해당 키워드를 가진 모든 파트너 카드 표시

### 사용자 경험
- 키워드 클릭 시 해당 키워드로 필터링된 목록 페이지로 이동
- "전체 보기" 버튼으로 필터 해제 및 전체 목록으로 돌아가기
- 키워드 태그에 hover 효과로 클릭 가능함을 표시
